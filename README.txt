# ASCII TV-Style Dashboard

Welcome to the **ASCII TV-Style Dashboard** — a retro-inspired, full-screen terminal dashboard built in Python.  
It combines system monitoring, animated hearts, and a scrolling story for a cozy digital experience.  

## Features

- 🕒 **Digital Clock** with tenths of seconds  
- ❤️ **Pulsing hearts** (`♥` and `<3`)  
- 📊 **System stats**:
  - CPU usage
  - RAM usage
  - Swap memory
  - Network traffic (up/down)
- 🎞️ **FPS counter** for smooth frame updates  
- 📜 **Scrolling story ticker** (left → right)  
- ✨ **Retro flicker effect** on bottom lines for “old TV” vibe  
- Full **80×20 terminal** support
